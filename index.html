Como jogar?

O Impostor — Guia rápido

01
Palavra secreta
Todos os detetives recebem a mesma palavra secreta. O impostor não recebe a palavra (ou recebe apenas uma similar, se o modo estiver ativado).
02
Rodadas de conversa
Os jogadores se revezam fazendo declarações ou perguntas relacionadas à palavra. Detetives dão pistas sem revelar a palavra. O impostor finge que sabe.
03
Votação
Após todos falarem, o grupo pode votar para eliminar um suspeito. Acertaram o impostor? Detetives vencem. Erraram? O jogo continua!
04
Chute do impostor
A qualquer momento, o impostor pode chutar a palavra secreta. Se acertar, vence sozinho. Se errar, os detetives ganham imediatamente.
05
Multi-impostor
Com 2+ impostores, o host configura o número antes do jogo. Impostores têm um chat secreto entre eles para se coordenar. Vencem se igualarem ou superarem os detetives vivos.
